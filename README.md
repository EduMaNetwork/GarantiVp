GarantiVp
=========

Garanti Bankası Sanal Pos İstemcisi .Net Framework üzerinde C# kullanılarak geliştirilmştir. Garanti Bankasının yeni 
sanal pos sistemine göre düzenlenmiştir. 

OOS_PAY ödeme tipi için düzenlenmiştir.
